Descripción del Proyecto

Este proyecto educativo interactivo busca visibilizar las “trampas invisibles” que operan dentro de las redes sociales: mecanismos de comparación, validación externa, algoritmos que manipulan lo que vemos e impactos en la salud mental. A través de dinámicas interactivas (tests, juegos, quizzes) el usuario no solo consume contenido, sino que también experimenta esas dinámicas y reflexiona sobre su relación con las redes.

Tecnologías Utilizadas

  • HTML5 – estructura semántica y accesible.
  • CSS3 – diseño responsivo, gradientes, efectos hover y animaciones suaves.
  • JavaScript (vanilla JS) – interactividad:
    • Quiz interactivo con botones “me gusta / no me interesa”.
    • Botones “Ver más / Ver menos”.
    • Control de visibilidad y temporizadores.
  • Flexbox, grid y `aspect-ratio` para diseño moderno.
  • Animaciones CSS y transiciones para mejor UX.

Conclusión

Este proyecto no busca dar respuestas cerradas, sino fomentar la reflexión. Conocer los mecanismos invisibles de las redes sociales permite cuestionar hábitos y promover un uso más consciente y equilibrado.
